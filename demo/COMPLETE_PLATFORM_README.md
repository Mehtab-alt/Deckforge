# DeckForge Complete Platform Evolution Demo

This demo folder contains a comprehensive demonstration of DeckForge's evolution through all 10 phases, showing how it transformed from a static infrastructure generator into an Event-Driven, Intent-Based Intelligence System.

## Contents

- `comprehensive_demo.py` - Complete demonstration of all 10 phases
- `visual_demo.py` - Visual simulation of Phases 9-10
- `interactive_demo.py` - Interactive demo allowing custom scenarios
- This README file
